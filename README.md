Project Manager Bot 
Телеграм-бот для управления персональными проектами. Помогает организовать и отслеживать ваши проекты, их статусы и необходимые навыки.
Основные функции
1.Создание проектов-сохранение названия, описания, ссылки и статуса

2.Отслеживание статусов-планирование, в процессе, завершены и др.

3.Управление навыками-добавление технологий и навыков к проектам

4.Редактирование-изменение информации о существующих проектах

5.Удаление-удаление ненужных проектов

6.Быстрый поиск-получение информации по названию проекта
