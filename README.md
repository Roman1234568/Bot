#Project Manager Bot 
Телеграм-бот для управления персональными проектами. Помогает организовать и отслеживать ваши проекты, их статусы и необходимые навыки.

Основные функции
Создание проектов - сохранение названия, описания, ссылки и статуса

Отслеживание статусов - планирование, в процессе, завершены и др.

Управление навыками - добавление технологий и навыков к проектам

Редактирование - изменение информации о существующих проектах

Удаление - удаление ненужных проектов

Быстрый поиск - получение информации по названию проекта
